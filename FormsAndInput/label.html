the <label> tag defines a label for several elements

Proper use of labels with the supported elements will benefit in:

  • Screen readers users (will read out loud the label, when the user is focused on the element)

  • Users who have difficulty clicking on very small regions (such as checkboxes) - because when a user clicks the text within the <label> element, it toggles the input (this increase the hit area)
