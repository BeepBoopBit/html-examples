The <s> tag displays a line in the middle of the text.

This tag is used specify texts that want to be seeen as incorrect or no longer relevant.

Example:

        <!DOCTYPE html>
            <html>
                <head>
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>S for incorrect</title>
                </head>
                <body>
                    <p>Example 1:</p>
                        <p>The elepgant elephant has a big trunk.</p>
                    <p>Example 2:</p>
                        <p>Only 50 tickets left!</p>
                        <p>SOLD OUT!</p>
                </body>
            </html>
