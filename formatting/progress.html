<progress> tag represent the completion progress of a task

Always add <label> tag for best accessibility practices

32%

Example:

        <!DOCTYPE html>
        <html>
            <head>
                <meta charset="UTF-8">
                <meta http-equiv="X-UA-Compatible" content="IE=edge">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
            </head>
            <body>
                <p>label for="file"</p> Downloading Progress: </label>
                <progress id="file" value="55" max=100> 55% </progress>
            </body> 
        </html>
