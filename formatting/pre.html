The <

> tag stands for "preformatted" text which presents the text exactly as written in the HTML source code.

    
This tag uses a fixed-width font, usually monospaced font.

    
It also displays the whitespace inside an element as written in the HTML code.

    
Example:

    
        <!DOCTYPE html>
            <html>
                <head>
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Preformatted Text</title>
                </head>
                <body>
                    <h1>Normal Text</h1>
                    <p>
                        Text in a pre element<br>
                        is displayed in a fixed-width<br>
                        font, and it preserves<br>
                        both      spaces and<br>
                        line breaks<br>
                    </p>
                    <h1>Using the <pre< tag</h1> 
                    <pre>
                        L         TE
                         A       A

                            REUSE
                           QUE TU
                        POR      TES
                    </pre>
                </body>
            </html>
