the <template>is used as a container to hold some HTML content hidden from the user when the page loads.

The content inside <template> can be rendered later with a javescript

You can use the <template> tag if you have some HTML code you want to use over and over again, but not until you ask for it.

To do this without <template> tag, you have to create the HTML code with JavaScript to prevent the browser from rendering the code

        <!DOCTYPE html>
        <html>
            <head>
                <meta charset="UTF-8">
                <meta http-equiv="X-UA-Compatible" content="IE=edge">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
            </head>
            <body>
                <button onclick="showContent()">Show Hidden Content</button>
                <template> This is hidden until the button is clicked</template>
            </body> 
            <script>
                function showContent(){
                    var temp = document.getElementsByTagName("template")[0];
                    var clon = temp.content.cloneNode(true);
                    document.body.appendChild(clon);
                }
            </script>
        </html>
