The <rp> tag can be used to provide parentheses around a ruby text, to be shown by browsers that do not support ruby annotations.

Use <rp> together with <ruby> and <rt>: the <ruby> element consists of one or more characters that needs an explanation/pronunciation, and an <rt> element that gives that information, and an opinion <rp> element that defines what to show for browsers that not support ruby annotations

漢 (ㄏㄢˋ) 
        <!DOCTYPE html>
        <html>
            <head>
                <meta charset="UTF-8">
                <meta http-equiv="X-UA-Compatible" content="IE=edge">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
            </head>
            <body>
            <ruby>
                漢 <rp>(</rp><rt>ㄏㄢˋ</rt><rp>)</rp>
            </ruby>
            </body> 
        </html>
