The <b> tag is used to bold a text.

Example: 

 
    <!DOCTYPE html>
    <html>
        <head>
            <meta charset="UTF-8">
            <meta http-equiv="X-UA-Compatible" content="IE=edge">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Bold</title>
        </head>
        <body>
            <p>This is normal text - <b>and this is bold text</b>.</p>
        </body>
    </html>
