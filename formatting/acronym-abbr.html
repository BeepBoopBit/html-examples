The tag is used to mark up an abbreviation or acryonym.

Note: There is also an tag used for HTML 4. The and tags are the same, but is used for HTML 5.

Example:

> 
    <!DOCTYPE html>
    <html>
        <head>
            <meta charset="UTF-8">
            <meta http-equiv="X-UA-Compatible" content="IE=edge">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>What is HTML?</title>
        </head>
        <body>
            <p> <abbr title="Hypertext Markup Language">HTML</abbr> is the standard markup language for creating Web pages.</p>
        </body>
    </html>
