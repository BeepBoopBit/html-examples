the <ins> tag defines a text that has been inserted into a document. Browsers will usually underline inserted text.

It supports, cite and datatime attribute

Example:

        <!DOCTYPE html>
        <html>
            <head>
                <meta charset="UTF-8">
                <meta http-equiv="X-UA-Compatible" content="IE=edge">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
            </head>
            <body>
                <p> The next word should be inserted, <ins>stuff</ins></p>
            </body> 
        </html>
