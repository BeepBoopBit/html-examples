The (Word Break Opportunity) tag specifies where in a text it would be ok to add a line-break.

This tag is similar to the
tag but is for browsers to line-break if there is an opportunity.

Note: When a word is too long, the browser might break it at the wrong place.

The wbr element is not supported in Internet Explorer 11 (or earlier).
