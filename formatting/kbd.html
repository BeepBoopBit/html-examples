<kbd> means keyboard input. Which is displayed in a sort of way depending on the browser

Example

        <!DOCTYPE html>
        <html>
            <head>
                <meta charset="UTF-8">
                <meta http-equiv="X-UA-Compatible" content="IE=edge">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
            </head>
            <body>
                <p>Press <kbd>Ctrl< + <kbd>C</kbd> to copy text</p>
            </body> 
        </html>
