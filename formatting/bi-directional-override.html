The tag stands for bi-directional override.

This tag is used to change the current direction of the text orientation.

This tag formats the text using:

"" - for "left to right" text orientation.

Example:

These are the nights that never die.

Output:

These are the nights that never die.

"" - for "right to left" text orientation.

Example:

These are the nights that never die.

Output:

These are the nights that never die.
