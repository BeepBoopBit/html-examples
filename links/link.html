The <link>tag defines the relationship between the current document and current document and an external resource.

The <link> tag is most often used to link to external style sheet or to add a favicon to your website.

The <link> element is an empty element, it contains attributes only.

        <!DOCTYPE html>
        <html>
            <head>
                <meta charset="UTF-8">
                <meta http-equiv="X-UA-Compatible" content="IE=edge">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <link rel="stylesheet" href="style.css">
            </head>
            <body>
            </body> 
        </html>
