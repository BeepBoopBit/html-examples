The <map> tag is used to define an image map. An image map is an image with clickable areas

The required name attribute of the <map> element is associated with the <img>'s use map attribute and creates a relationship between the image and the map

The <map> element contains a number of <area> elements, that defines the clickable areas in the image map.

        <!DOCTYPE html>
        <html>
            <head>
                <meta charset="UTF-8">
                <meta http-equiv="X-UA-Compatible" content="IE=edge">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
            </head>
            <body>
            <map name="workmap">
                <area shape="rect" coords="34,44,270,340" href="https://google.com" alt="google">
                <area shape="rect" coords="290,172,333,250" href="https://youtube.com" alt="youtube">
                <area shape="circle" coords="337,300,44" href="https://facebook.com" alt="facebook">
            </map>
            </body> 

        </html>
