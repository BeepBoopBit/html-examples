the <head> contains the metadata of the documents

Metadata are data about data.

Example:

When you have a file a look to its properties, you will see when the file is created, it's allocated memory and the such. Those are metadata.

        <!DOCTYPE html>
        <html>
            <head>
                <meta charset="UTF-8">
                <meta http-equiv="X-UA-Compatible" content="IE=edge">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
            </head>
            <body>

            </body> 

        </html>
